# Sequoia for iTerm


# Installation

1. Downloads the files from this repository with the extension ".itermcolors"
2. Open iTerm.
3. Navigate to Prefferences>Profile>Colors
4. Under the color presects menu choose import.
5. Import the theme you want to use.
6. Enjoy.


# Avilable themes
- Moonlight
- Monochrome
